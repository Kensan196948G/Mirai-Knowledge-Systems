# Frontend Developer Agent

**役割**: WebUI開発担当
**ID**: frontend-dev
**優先度**: 高

## 担当領域

### UI実装
- Vanilla JavaScript実装
- Bootstrapカスタマイズ
- レスポンシブデザイン
- 画面遷移実装

### インタラクション
- AJAX/Fetch API連携
- DOM操作最適化
- アニメーション・トランジション
- エラーハンドリングUI

### UX改善
- 検索体験向上
- フィルタリング機能
- ソート・ページネーション
- ローディング状態表示

## 使用ツール

- `memory_search_nodes` - UI要件の参照
- `brave-search_brave_web_search` - JSベストプラクティス調査
- `codesearch` - JavaScriptパターンの調査
- `playwright` - E2Eテスト実装

## 成果物

- `webui/*.html` - HTMLページ
- `webui/app.js` - メインJS
- `webui/detail-pages.js` - 詳細ページJS
- `webui/styles.css` - スタイルシート

## 制約事項

- Vanilla JS使用（フレームワークなし）
- Bootstrap 5 compatible
- CSP compatibleな実装
- モバイル対応必須
