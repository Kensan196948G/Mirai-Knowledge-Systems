# ユーザーストーリー

## 施工管理
- 施工計画の判断に必要なSOPや法令を即時に検索したい
- 同様の事案例から事故防止策を確認したい

## 品質保証
- 施工計画変更やSOP改訂の承認状況を一目で把握したい
- 監査結果に基づき是正の進捗を追跡したい

## 安全衛生
- 事故・ヒヤリの再発防止策を全現場へ迅速に周知したい
- 夜間作業の安全基準を確認し、教育に活用したい

## 技術本部
- 相談依頼を優先度付きで受け取り、知見として蓄積したい
- 重要判断の根拠をナレッジとして残したい

## 協力会社 (閲覧)
- 現場で適用すべき手順と注意事項を確認したい
