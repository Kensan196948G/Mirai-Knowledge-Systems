# ユースケース

## UC-01 ナレッジ検索
- 主アクター: 施工管理
- トリガー: 施工判断の確認
- 事前条件: 権限付与済み
- 成果: 関連SOP/法令/事故レポートを閲覧

## UC-02 承認フロー更新
- 主アクター: 所長、品質保証
- トリガー: 施工計画変更の提出
- 成果: 承認ステータスが更新され配信準備へ移行

## UC-03 事故レポートの配信
- 主アクター: 安全衛生
- トリガー: 事故レポート承認完了
- 成果: 全現場への通知と閲覧ログ記録

## UC-04 専門家相談
- 主アクター: 現場代理人
- トリガー: 判断が難しい施工条件の発生
- 成果: 回答がナレッジ化され現場へ共有
