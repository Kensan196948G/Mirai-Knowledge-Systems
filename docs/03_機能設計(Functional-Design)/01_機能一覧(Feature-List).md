# 機能一覧

## ナレッジ管理
- 種別別の登録・更新・閲覧
- タグ付与、関連付け、改訂履歴

## 検索・推薦
- 条件検索 (工種/工区/発注者/更新日)
- 関連ナレッジの自動提示

## 承認・配信
- 多段承認フロー
- 配信先の制御と配信履歴の保存

## 事故・ヒヤリ
- 事案登録、原因分析、再発防止策
- 重要アラートの提示

## 専門家相談
- 相談起案、専門家アサイン、回答配信

## 監査・レポート
- 参照/変更ログ
- 進捗・稼働ダッシュボード
