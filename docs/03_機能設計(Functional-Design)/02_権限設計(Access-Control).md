# 権限設計

## 役割
- 管理者: 全機能、設定、監査ログ閲覧
- 所長: 承認、配信、現場データ更新
- 施工管理: 登録・更新・閲覧
- 品質保証/安全衛生: 承認、レビュー、配信
- 協力会社: 閲覧のみ

## 権限マトリクス (抜粋)
| 機能 | 管理者 | 所長 | 施工管理 | 品質/安全 | 協力会社 |
| --- | --- | --- | --- | --- | --- |
| ナレッジ閲覧 | ○ | ○ | ○ | ○ | ○ |
| ナレッジ登録 | ○ | ○ | ○ | ○ | - |
| 承認 | ○ | ○ | - | ○ | - |
| 配信 | ○ | ○ | - | ○ | - |
| 監査ログ閲覧 | ○ | ○ | - | ○ | - |

## 運用ルール
- 役割変更は管理者のみが実施
- 協力会社は閲覧範囲を現場単位で限定
