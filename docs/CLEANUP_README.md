## 📚 ドキュメント構成

このクリーンアップ計画は4つのドキュメントで構成されています。

### 1. CLEANUP_SUMMARY.md（エグゼクティブサマリー）⭐ 最初に読む

**読了時間**: 5分
**内容**:

- 最優先アクション（今日中）

## ⚠️ 重要な注意事項

### マージ前の確認

- CI/CD全体が通過していること
- コンフリクトが解消されていること
- レビューが完了していること

### ブランチ削除前の確認

- PRが完全にマージまたはクローズされていること
- ローカルとリモートの同期確認
- 削除後の復旧が不要であることを確認

### 緊急時のロールバック

- マージ後に問題が発生した場合は即座にrevert
- ブランチ削除は git reflog で復旧可能（30日間）
