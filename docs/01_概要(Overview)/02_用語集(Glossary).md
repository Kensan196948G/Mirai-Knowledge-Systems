# 用語集

| 用語 | 説明 |

| ナレッジ | 施工計画・品質・安全・原価・出来形に関する知見全般 |
| SOP | 標準作業手順 (Standard Operating Procedure) |
| 事故・ヒヤリ | 事故、ヒヤリハット事案 |
| 承認フロー | ナレッジ公開前のレビューと承認の流れ |
| 配信 | 承認後の現場への周知・通知 |
| 関連現場 | ナレッジが適用される現場 |
| 専門家相談 | 技術本部などへの判断支援リクエスト |
| 監査ログ | 参照・更新・承認等の操作記録 |
| SLA | 相談回答や通知配信のサービスレベル |
