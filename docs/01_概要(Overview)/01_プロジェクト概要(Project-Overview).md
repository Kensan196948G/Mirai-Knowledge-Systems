# プロジェクト概要

## 目的
- 施工計画・品質・安全・原価・出来形の知見を統合し、現場判断の速度と品質を高める
- 事例・SOP・法令・事故レポートを一元化し、再利用率を向上させる

## 背景
- 現場ごとの知見が分散し、共有に時間がかかる
- 法令改訂やSOP変更の周知にラグがある
- 事故・ヒヤリの再発防止ナレッジが現場間で横展開できていない

## 目標・KPI
- ナレッジ再利用率: 70%以上
- 事故ゼロ継続日数の維持
- 承認フローのリードタイム: 30%短縮
- 重大ナレッジの現場周知: 48時間以内

## 想定利用部門
- 施工管理
- 品質保証
- 安全衛生
- 技術本部
- 協力会社 (閲覧中心)

## 成果物
- 要件定義書、機能設計、非機能要件、データ設計、UI/UX設計
- 画面プロトタイプ (webui)
- テスト計画・受入基準
- 運用・保守手順

## スコープ
### インスコープ
- ナレッジ検索、閲覧、配信、承認フロー
- SOP、法令、事故レポート、専門家相談
- 監査ログと周知履歴

### アウトオブスコープ
- 現場日報や勤怠の一次登録
- 施工機材のIoT制御
- ERP/会計の詳細原価計算
