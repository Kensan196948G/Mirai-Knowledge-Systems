## 1. はじめに

### 1.1 Mirai Knowledge Systemsとは

Mirai Knowledge Systemsは、建設土木現場での迅速な判断を支援するナレッジマネジメントシステムです。このシステムを使うと、以下のことができます：

- SOP（標準施工手順）の検索と閲覧
- 法令・規格の確認
- 過去の事故レポートの参照
- 専門家への相談
- 現場で得た知見の共有

![システム概要](images/user/system-overview.png)

### 1.2 このマニュアルの対象者

このマニュアルは、以下の役割のユーザーを対象としています：

| 役割 | 説明 | できること |
|------|------|-----------|
| **施工管理** | 現場代理人、主任技術者 | ナレッジ検索、作成、相談投稿 |
| **安全衛生担当** | 安全衛生責任者 | 事故レポート作成、配信 |
| **技術専門家** | 本社技術部門 | 相談回答、ナレッジ作成 |
| **閲覧者** | 協力会社 | 公開ナレッジの閲覧のみ |

### 1.3 推奨動作環境

| 項目 | 推奨 |
|------|------|
| **ブラウザ** | Google Chrome 最新版、Firefox 最新版、Edge 最新版 |
| **画面解像度** | 1366x768 以上 |
| **インターネット接続** | 常時接続（オフラインモードは非対応） |
| **OS** | Windows 10以降、macOS 10.15以降、Linux（Ubuntu 20.04以降） |

## 2. ログインとログアウト

### 2.1 初回ログイン

#### 手順

1. ブラウザで以下のURLにアクセス：
   ```
   https://knowledge.example.com
   ```
   (実際のURLは管理者から通知されます)

2. ログイン画面が表示されます

![ログイン画面](images/user/login-screen.png)

3. 管理者から通知されたユーザー名とパスワードを入力
   - ユーザー名: `yamada_taro` (例)
   - パスワード: 初期パスワード

4. **「ログイン」** ボタンをクリック

5. 初回ログイン時は **パスワード変更画面** が表示されます

![初回パスワード変更](images/user/first-password-change.png)

6. 新しいパスワードを設定：
   - 8文字以上
   - 英大文字、英小文字、数字、記号を各1文字以上含む
   - 例: `MyPass123!`

7. **「パスワード変更」** ボタンをクリック

8. ダッシュボードが表示されれば成功です

### 2.2 通常ログイン

#### 手順

1. ログイン画面でユーザー名とパスワードを入力
2. **「ログイン」** ボタンをクリック
3. ダッシュボードが表示されます

#### ログインできない場合

以下を確認してください：

- [ ] ユーザー名は正しいですか？（大文字・小文字を区別します）
- [ ] パスワードは正しいですか？
- [ ] Caps Lockがオンになっていませんか？
- [ ] ブラウザのキャッシュをクリアしましたか？

それでもログインできない場合は、管理者に連絡してパスワードリセットを依頼してください。

### 2.3 ログアウト

#### 手順

1. 画面右上のユーザー名をクリック
2. ドロップダウンメニューから **「ログアウト」** を選択
3. ログイン画面に戻ります

![ログアウトメニュー](images/user/logout-menu.png)

**重要**: 共有PCを使用している場合は、必ずログアウトしてください。

## 4. ナレッジの検索

### 4.1 基本検索

#### 手順

1. ヘッダーの **検索バー** にキーワードを入力
   - 例: `コンクリート 養生`
2. Enterキーを押すか、虫眼鏡アイコンをクリック
3. 検索結果が表示されます

![基本検索](images/user/basic-search.png)

### 4.2 詳細検索

より詳細な条件で検索できます：

#### 手順

1. サイドバーから **「ナレッジ検索」** を選択
2. **「詳細検索」** をクリック

![詳細検索画面](images/user/advanced-search.png)

3. 検索条件を設定：

| 条件 | 説明 | 例 |
|------|------|-----|
| **キーワード** | タイトルや本文に含まれる語句 | `配筋検査` |
| **カテゴリ** | ナレッジの種類 | SOP、法令、事故レポート、相談 |
| **タグ** | 関連タグ（複数選択可） | `安全`, `品質管理` |
| **優先度** | 重要度 | 高、中、低 |
| **日付範囲** | 作成日または更新日 | 2024-01-01 〜 2024-12-31 |
| **プロジェクト** | 特定のプロジェクト | `〇〇橋梁工事` |

4. **「検索」** ボタンをクリック

### 4.3 検索結果の見方

検索結果には以下の情報が表示されます：

- **タイトル**: ナレッジのタイトル
- **概要**: 簡単な説明
- **カテゴリ**: 種別（SOP、法令など）
- **タグ**: 関連タグ
- **更新日**: 最終更新日
- **作成者**: 作成したユーザー
- **ステータス**: 承認済み、承認待ちなど

![検索結果リスト](images/user/search-results.png)

#### 並び替え

検索結果は以下の順で並び替えられます：

- **関連度順**（デフォルト）: キーワードとの関連性が高い順
- **更新日順**: 最新のものが上
- **作成日順**: 新しく作成されたものが上
- **閲覧数順**: よく見られているものが上

### 4.4 検索のコツ

#### AND検索（複数キーワード）

スペースで区切ると、両方を含む結果が表示されます。

```
コンクリート 養生
→ 「コンクリート」AND「養生」を含む
```

#### 完全一致検索

ダブルクォートで囲むと、そのフレーズを完全一致で検索します。

```
"打設後24時間"
→ 正確にこのフレーズを含む
```

#### 除外検索

マイナス記号で特定のキーワードを除外できます。

```
コンクリート -舗装
→ 「コンクリート」を含むが「舗装」を含まない
```

## 5. ナレッジの作成

### 5.1 ナレッジ作成の流れ

施工管理や技術専門家の方は、新しいナレッジを作成できます。

```
ナレッジ作成 → 下書き保存 → 承認申請 → 承認者確認 → 承認 → 公開
```

### 5.2 ナレッジ作成手順

#### 手順

1. サイドバーから **「ナレッジ検索」** を選択
2. 右上の **「新規作成」** ボタンをクリック

![新規作成ボタン](images/user/create-knowledge-button.png)

3. ナレッジ作成フォームが表示されます

![ナレッジ作成フォーム](images/user/knowledge-create-form.png)

4. 必須項目を入力：

**基本情報**

| 項目 | 説明 | 入力例 |
|------|------|-------|
| **タイトル** | わかりやすいタイトル（必須） | `コンクリート打設時の養生方法` |
| **概要** | 内容の要約（必須、200文字以内） | `夏期高温時のコンクリート打設後の適切な養生手順について説明` |
| **カテゴリ** | ナレッジの種類（必須） | SOP、法令、事故レポート、相談回答 |
| **タグ** | 関連キーワード（複数可） | `コンクリート`, `養生`, `夏期施工` |
| **優先度** | 重要度 | 高、中、低 |
| **プロジェクト** | 関連プロジェクト（任意） | `〇〇橋梁工事` |

**詳細内容**

| 項目 | 説明 |
|------|------|
| **本文** | 詳細な説明（Markdown記法が使えます） |

**Markdown記法の例**:

```markdown
## 手順

1. 打設完了後、直ちに表面をビニルシートで覆う
2. 散水養生を開始する
   - 頻度: 2時間ごと
   - 期間: 最低7日間

### 注意事項

- 気温35℃以上の場合は散水頻度を1時間ごとに変更
- 直射日光を避けるため、シートは白色推奨

**参考法令**: 土木工事共通仕様書 第3編 第2章
```

5. **添付ファイル**（任意）
   - **「ファイル追加」** ボタンをクリック
   - 画像、PDF、Excelなどをアップロード
   - 最大サイズ: 10MB/ファイル

![添付ファイルアップロード](images/user/knowledge-attachment.png)

6. **「下書き保存」** または **「承認申請」** を選択

### 5.3 下書き保存と承認申請の違い

| 操作 | 状態 | 説明 |
|------|------|------|
| **下書き保存** | `draft` | 他のユーザーには見えない。後で編集可能 |
| **承認申請** | `pending` | 承認者に通知され、確認待ち状態になる |

### 5.4 承認申請後の流れ

1. 承認申請をクリック
2. 承認者（所長、品質保証室など）に通知が送信されます
3. 承認者が内容を確認
4. **承認** されると、ステータスが `approved` になり、全ユーザーに公開
5. **却下** された場合は、却下理由が通知されます。修正して再申請可能

![承認フロー](images/user/approval-flow.png)

### 5.5 自分のナレッジ管理

#### マイページで確認

1. サイドバーから **「マイページ」** を選択
2. 自分が作成したナレッジ一覧が表示されます

![マイページ](images/user/my-page.png)

#### ステータス別表示

- **下書き**: 編集継続可能
- **承認待ち**: 承認者の確認待ち
- **承認済み**: 公開中
- **却下**: 却下理由を確認し、修正して再申請

#### ナレッジの編集

1. マイページで編集したいナレッジをクリック
2. **「編集」** ボタンをクリック
3. 内容を修正
4. **「保存」** または **「承認申請」** をクリック

**注意**: 承認済みのナレッジを編集すると、再度承認が必要になります。

## 6. SOPの閲覧

### 6.1 SOPとは

SOP（Standard Operating Procedure）は、標準施工手順書です。会社が定めた施工の標準的な手順が記載されています。

### 6.2 SOP一覧の表示

#### 手順

1. サイドバーから **「SOP」** を選択
2. SOP一覧が表示されます

![SOP一覧](images/user/sop-list.png)

### 6.3 SOPのフィルタリング

以下の条件でフィルタできます：

| 条件 | 説明 | 例 |
|------|------|-----|
| **カテゴリ** | 工種別 | 土工、コンクリート工、鋼構造物工 |
| **バージョン** | 版数 | Ver.1.0, Ver.2.0 |
| **ステータス** | 有効/無効 | 有効、廃止、改訂中 |
| **タグ** | 関連キーワード | 安全、品質、環境 |

![SOPフィルタ](images/user/sop-filter.png)

### 6.4 SOP詳細の閲覧

#### 手順

1. SOP一覧から閲覧したいSOPをクリック
2. SOP詳細画面が表示されます

![SOP詳細](images/user/sop-detail.png)

#### SOP詳細画面の構成

- **タイトル**: SOPの名称
- **カテゴリ**: 工種分類
- **バージョン**: 現在の版数
- **改訂日**: 最終改訂日
- **対象**: 適用対象（例: 全現場、特定工事種別）
- **本文**: 手順の詳細
- **添付ファイル**: 図面、写真、チェックリストなど
- **改訂履歴**: 過去の版との違い

### 6.5 SOPのダウンロード

PDF形式でダウンロードできます：

1. SOP詳細画面で **「PDF出力」** ボタンをクリック
2. ブラウザのダウンロードフォルダに保存されます

![SOPダウンロード](images/user/sop-download.png)

**用途**: 現場で印刷して掲示、オフライン参照など

### 6.6 改訂履歴の確認

SOPは定期的に改訂されます。変更内容を確認できます：

1. SOP詳細画面で **「改訂履歴」** タブをクリック
2. 過去の版が一覧表示されます
3. 各版をクリックすると内容を確認できます

![SOP改訂履歴](images/user/sop-revision-history.png)

## 7. 通知の確認

### 7.1 通知の種類

システムから以下の通知が届きます：

| 通知タイプ | 内容 | 対象者 |
|-----------|------|-------|
| **承認完了** | 自分が申請したナレッジが承認された | 申請者 |
| **承認却下** | 自分が申請したナレッジが却下された | 申請者 |
| **新規SOP公開** | 新しいSOPが公開された | 全ユーザー |
| **SOP改訂** | SOPが改訂された | 全ユーザー |
| **事故レポート** | 重要な事故レポートが公開された | 全現場責任者 |
| **専門家相談回答** | 自分の相談に回答があった | 相談者 |
| **重要通知** | 管理者からの緊急連絡 | 指定ユーザー |

### 7.2 通知の確認方法

#### ヘッダーの通知アイコン

1. ヘッダー右上の **ベルアイコン** に未読数が表示されます
2. アイコンをクリックすると、通知リストが表示されます

![通知アイコン](images/user/notification-icon.png)

3. 通知をクリックすると詳細画面に移動します

![通知リスト](images/user/notification-list.png)

#### 通知センター

全ての通知を確認できます：

1. 通知リスト下部の **「すべて見る」** をクリック
2. 通知センターが表示されます

![通知センター](images/user/notification-center.png)

### 7.3 通知のフィルタリング

通知センターでは以下でフィルタできます：

- **未読のみ**: 未読通知だけを表示
- **既読のみ**: 既読通知だけを表示
- **種類別**: 承認、SOP、事故レポートなど

### 7.4 通知の既読/未読管理

#### 既読にする

- 通知をクリックすると自動的に既読になります
- または、通知の右側の **「既読にする」** ボタンをクリック

#### 一括既読

1. 通知センターで **「すべて既読にする」** ボタンをクリック
2. 全ての未読通知が既読になります

![一括既読](images/user/mark-all-as-read.png)

### 7.5 通知設定

不要な通知を受け取らないように設定できます：

1. ヘッダー右上の **ユーザーメニュー** → **「設定」** をクリック
2. **「通知設定」** タブを選択
3. 受け取りたい通知をチェックボックスで選択
4. **「保存」** ボタンをクリック

![通知設定](images/user/notification-settings.png)

#### 設定可能な項目

- [ ] 承認完了通知
- [ ] 承認却下通知
- [ ] 新規SOP公開通知
- [ ] SOP改訂通知
- [ ] 事故レポート通知
- [ ] 専門家相談回答通知
- [ ] 重要通知（変更不可）

## 8. よくある質問（FAQ）

### 8.1 ログイン・認証

**Q1: パスワードを忘れました。どうすればいいですか？**

A1: ログイン画面の **「パスワードを忘れた場合」** リンクをクリックし、メールアドレスを入力してください。パスワードリセット用のリンクが送信されます。それでも解決しない場合は、管理者に連絡してください。

**Q2: ログインしようとすると「アカウントがロックされています」と表示されます。**

A2: 5回連続でパスワードを間違えると、30分間アカウントがロックされます。30分経過後に再度試すか、管理者に連絡してロック解除を依頼してください。

**Q3: セッションタイムアウトのメッセージが頻繁に表示されます。**

A3: 30分間操作がないと自動的にログアウトされます。作業中は定期的に画面を操作してください。長時間離席する場合は、一度ログアウトすることをお勧めします。

### 8.2 検索

**Q4: 検索しても目的のナレッジが見つかりません。**

A4: 以下を試してください：
- キーワードを変えてみる（例: `養生` → `キュアリング`）
- 詳細検索でカテゴリやタグを指定する
- 日付範囲を広げる
- 下書きや承認待ちのナレッジは検索結果に表示されません

**Q5: 削除されたナレッジを見ることはできますか？**

A5: 削除されたナレッジは検索結果に表示されません。管理者に問い合わせれば、監査ログから削除前の情報を確認できる場合があります。

### 8.3 ナレッジ作成

**Q6: ナレッジを作成中に誤って画面を閉じてしまいました。データは失われますか？**

A6: 自動保存機能により、5分ごとに下書きが保存されます。再度ログインしてマイページを確認してください。下書きとして残っている可能性があります。

**Q7: 添付できるファイルの種類は何ですか？**

A7: 以下の形式に対応しています：
- 画像: JPG, PNG, GIF, BMP（各10MBまで）
- ドキュメント: PDF, Word, Excel, PowerPoint（各10MBまで）
- その他: テキスト、CSV、ZIP（各10MBまで）

**Q8: 承認申請後にミスを見つけました。取り下げできますか？**

A8: マイページから該当ナレッジを選択し、**「承認申請取り下げ」** ボタンをクリックしてください。ステータスが下書きに戻り、編集できます。

**Q9: 却下されたナレッジはどうすればいいですか？**

A9: 却下理由を確認し、内容を修正して再度承認申請してください。却下理由は通知またはナレッジ詳細画面で確認できます。

### 8.4 権限

**Q10: 他のユーザーのナレッジを編集できますか？**

A10: 基本的に自分が作成したナレッジのみ編集できます。管理者と承認者は全てのナレッジを編集できます。

**Q11: SOPを編集したいのですが、編集ボタンが表示されません。**

A11: SOPは技術本部と管理者のみが編集できます。修正が必要な場合は、専門家相談から修正依頼を投稿してください。

### 8.5 通知

**Q12: 通知が届きません。**

A12: 以下を確認してください：
- 通知設定で該当の通知がオンになっているか
- ユーザー情報のメールアドレスが正しいか
- 迷惑メールフォルダに振り分けられていないか
- それでも解決しない場合は管理者に連絡

**Q13: 通知が多すぎて困っています。**

A13: 通知設定で不要な通知をオフにできます。ヘッダーメニュー → **「設定」** → **「通知設定」** から変更してください。

### 8.6 その他

**Q14: スマートフォンやタブレットでも使えますか？**

A14: 基本的な機能は使えますが、快適な操作にはPC推奨です。特にナレッジ作成や詳細検索はPCでの利用をお勧めします。

**Q15: システムの動作が遅いです。**

A15: 以下を試してください：
- ブラウザのキャッシュをクリアする
- 他のタブを閉じる
- ブラウザを再起動する
- それでも改善しない場合は管理者に連絡

**Q16: オフラインでも使えますか？**

A16: 現在、オフラインモードには対応していません。インターネット接続が必要です。

**Q17: データをエクスポートできますか？**

A17: SOPや一部のナレッジはPDF形式でダウンロードできます。大量データのエクスポートが必要な場合は管理者に相談してください。
