# 運用手順

本書は日次・週次・月次の運用手順をまとめたチェックリストです。実運用では `docs/11_運用保守(Operations)/11_運用スクリプト一覧(Operations-Scripts).md` を併用してください。

## 日次

- サービス稼働確認（systemd / Gunicorn / Nginx）
- 監視ダッシュボード確認（アラート有無、主要メトリクス）
- エラーログ確認（APIエラー、認証失敗、5xx急増）
- 承認待ち・重要通知の確認

## 週次

- バックアップの取得状況確認
- ログローテーションの実行/結果確認
- セキュリティスキャン結果の確認
- データ増加状況と検索品質のレビュー

## 月次

- SLA/SLOレビューと改善項目整理
- 権限棚卸し（管理者/協力会社アカウント）
- パスワード運用の見直し（期限/失効）
- ストレージ使用量の見直しと増設判断
- 証明書有効期限の確認
