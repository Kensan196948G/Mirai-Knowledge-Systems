## Partner Access Overview

### Access Scope

External partners have **read-only access** to:
- ✅ Approved knowledge entries
- ✅ Safety procedures and incident reports
- ✅ Standard operating procedures (SOPs)
- ✅ Regulatory and compliance information
- ✅ Public project information

### Access Restrictions

Partners **cannot**:
- ❌ Create or edit knowledge content
- ❌ Submit incident reports
- ❌ Request expert consultations
- ❌ Access draft or pending content
- ❌ Modify system settings or user accounts

### Account Types

| Account Type | Access Level | Purpose |
|-------------|-------------|---------|
| **Partner** | Read-only | General access to approved content |
| **Subcontractor** | Limited read-only | Project-specific content only |
| **Supplier** | Material-specific | Content related to supplied materials |

## Reporting Issues

### Content Issues

#### Reporting Inaccurate Information
If you find content that appears incorrect or outdated:

1. **Document the Issue**
   - Note the content title and location
   - Describe the specific problem
   - Reference correct information if known

2. **Report to Administrator**
   - Use the "問題報告" (Report Issue) button
   - Select "コンテンツの問題" (Content Issue)
   - Provide detailed description

3. **Follow Up**
   - Check for updates to the reported content
   - Contact your company representative if urgent

### System Issues

#### Technical Problems
Report technical issues such as:
- Login problems
- Search not working
- Content not loading
- Display errors

#### Reporting Process
1. **Gather Information**
   - Browser type and version
   - Error messages (if any)
   - Steps to reproduce the issue
   - Screenshot if possible

2. **Submit Report**
   - Use "システムの問題" (System Issue) in report form
   - Include all gathered information
   - Specify urgency level
