# Mirai Knowledge Systems クイックスタートカード

## 🚀 5分で始める

### ステップ1: ログイン

```
URL: https://knowledge.example.com
ユーザー名: 管理者から通知された名前
パスワード: 初期パスワード
```

### ステップ2: ダッシュボード確認

ログイン後、以下を確認：
- 📢 **重要な通知** - 新着情報
- 📝 **承認待ち** - 自分の投稿状況
- 🕐 **最近閲覧** - 履歴へのクイックアクセス

### ステップ3: 検索

**基本検索**: ヘッダーの検索バーにキーワード入力

**検索のコツ**:
- 複数キーワード → スペースで区切る
- 完全一致 → `"フレーズ"`
- 除外 → `-キーワード`

---

## 📱 主要メニュー

| メニュー | 機能 |
|---------|------|
| 🏠 ダッシュボード | ホーム画面 |
| 🔍 ナレッジ検索 | 情報を探す |
| 📋 SOP | 標準手順書 |
| ⚠️ 事故レポート | 過去事例 |
| 💬 専門家相談 | 相談投稿 |
| 👤 マイページ | 自分の投稿 |

---

## ✏️ ナレッジ作成（3ステップ）

1. **ナレッジ検索** → **新規作成**
2. タイトル・内容を入力
3. **下書き保存** or **承認申請**

---

## 🔔 通知確認

ヘッダー右上の **🔔 ベルアイコン** をクリック

---

## ❓ 困ったら

- **FAQ**: ユーザーガイド > FAQ
- **サポート**: support@example.com
- **緊急**: emergency@example.com

## ⌨️ ショートカット

| キー | 機能 |
|------|------|
| `Ctrl+K` | 検索 |

| `Ctrl+S` | 保存 |
| `Esc` | ダイアログを閉じる |
