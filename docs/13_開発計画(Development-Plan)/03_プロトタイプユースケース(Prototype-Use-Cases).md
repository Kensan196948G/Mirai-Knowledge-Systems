# プロトタイプユースケース

## 選定方針
- 現場判断に直結するナレッジの検索・閲覧
- 承認と配信の流れが理解できること
- 事故・ヒヤリの共有と再発防止の流れが確認できること
- 専門家相談の起案から回答までの流れが確認できること

## 対象ユースケース一覧
### PU-01 ナレッジ検索と閲覧
- 概要: 工種/工区/発注者で検索し、詳細を閲覧する
- 成功条件: 主要情報が3クリック以内で到達できる

### PU-02 承認フローの確認
- 概要: 施工計画変更の承認状況を把握する
- 成功条件: ステータスと次の担当が明確に表示される

### PU-03 事故レポートの参照
- 概要: 事故の概要、原因、再発防止策を確認する
- 成功条件: 再発防止策が一覧で把握できる

### PU-04 法令改訂の影響確認
- 概要: 改訂ポイントと影響現場を確認する
- 成功条件: 影響範囲と対応タスクが表示される

### PU-05 SOPの点検手順確認
- 概要: SOPの手順、判定基準、履歴を確認する
- 成功条件: 手順と判定基準が同一画面で確認できる

### PU-06 専門家相談の起案
- 概要: 相談内容を入力し、必要資料を確認する
- 成功条件: 相談フォームと必要資料が同一画面で確認できる

## レビュー評価の対象
- ユースケースの網羅性
- 画面構成の妥当性
- 操作導線の分かりやすさ
