# ナビゲーション設計

## 目的
- 主要画面への到達性を高め、迷わない導線を提供する

## ナビゲーション構成
### グローバルナビゲーション (ヘッダー)
- 現場ダッシュボード
- 検索結果詳細
- SOP詳細
- 法令詳細
- 事故レポート
- 専門家相談

### ローカルナビゲーション (タブ/フィルタ)
- タブ: 検索結果/標準施工/事故・ヒヤリ
- フィルタ: 工種/工区/発注者/更新日

### 補助ナビゲーション (サイドバー)
- ナレッジカテゴリ
- プロジェクト
- 専門家/関連SOP

## 導線ルール
- ヘッダーは常時表示し、主要画面へ1クリックで遷移
- タブは同一画面内の情報切替に限定
- フィルタは複数選択可能

## 画面間遷移の基本
- ダッシュボード → 詳細画面 (クリックで遷移)
- 詳細画面 → 関連ナレッジ (右レールから遷移)

## 成功条件
- 主要画面が3クリック以内に到達できる
- 画面内操作で迷いがない
