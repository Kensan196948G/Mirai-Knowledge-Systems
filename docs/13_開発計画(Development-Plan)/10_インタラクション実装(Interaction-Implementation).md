# インタラクション実装

## 目的
- 画面内の主要操作を直感的に行えるようにする

## 実施内容
- タブ切替時の状態制御 (選択状態の明確化)
- フィルタ/チップの選択状態を維持
- 進捗バーの表示アニメーション

## 実装差分
- `webui/app.js`
  - タブ切替時の `aria-selected` 反映
  - フィルタ/チップの `aria-pressed` 自動設定
  - 進捗バーの値反映

## 成果物
- `webui/app.js`

## レビュー評価観点
- 操作に対する反応速度
- 状態の分かりやすさ
- 期待通りの挙動
